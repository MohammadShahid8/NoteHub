NoteHub

NoteHub is a simple yet powerful note-taking web application that allows users to create, view, and search for notes. It has a clean, beautiful homepage, a login and registration page, and intuitive features to manage your notes. Built using Java, Servlet, MySQL, Hibernate, HTML, CSS, and JSP, NoteHub is the perfect solution for users to organize and access their notes effortlessly.



Features

Beautiful Homepage: A visually appealing home page with an image to introduce the app.
Login and Register: Users can register for a new account and log in to access their notes securely.
Add Notes: Users can easily add notes with a title, description, and content.
View Notes: A user-friendly interface to view all the saved notes.
Search Notes: Users can search for their notes by title or content for quick access.



Technologies Used

Java: The core programming language for the backend logic.
Servlets: Used to handle HTTP requests and manage the business logic of the application.
JSP: JavaServer Pages for dynamic content generation.
HTML/CSS: Used for structuring and styling the front-end.
MySQL: A relational database to store user and note data.
Hibernate: ORM (Object-Relational Mapping) for database interactions and managing data persistence.
